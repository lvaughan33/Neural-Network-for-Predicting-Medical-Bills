This project predicts individual health insurance charges from demographic information, including age, sex, BMI, number of children, smoking status, and region. We begin with exploratory data analysis to verify data quality, inspect feature types, and visualize the target variable distribution. Categorical variables are converted to numerical features using one-hot encoding, while both the input features and target values are normalized to improve training stability. A feedforward neural network built with TensorFlow/Keras is then trained using two hidden layers (128 and 64 neurons) with ReLU activations, batch normalization, dropout, and L2 regularization to reduce overfitting. Model performance is evaluated on separate validation and test sets using mean squared error (MSE) and mean absolute error (MAE), achieving low prediction error and demonstrating strong generalization. Additional visualizations of the training history, predictions, and feature importance show that smoking status, BMI, and age are the most influential factors in predicting medical insurance costs.
